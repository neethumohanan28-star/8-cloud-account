## Cloud Computing

Cloud computing means storing and accessing data and programs over the Internet instead of computer’s hard drive or a local servers. 
Its services are provided on demand that can be accessed from anywhere at anytime and can be scalable. 
Cloud computing follows a pay-as-you-use pricing model, helping reduce capital and operational costs.
The wide ranges of cloud services are virtual servers, storage systems, databases, networking, security tools, and software applications, which can be used for personal, academic, or business purposes.
The main cloud service models are IaaS, PaaS, and SaaS, each offering different levels of control and management.
